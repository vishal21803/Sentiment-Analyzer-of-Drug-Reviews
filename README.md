# Drug Review Sentiment Analyzer

This is a Flask-based web application that analyzes drug reviews using a custom HuggingFace transformer model and predicts sentiment and rating.

---

## Features

- Analyze sentiment of drug reviews (Positive / Neutral / Negative)
- Provides a rating (1â€“5) based on confidence
- Shows confidence score
- Easy to extend for other NLP tasks
